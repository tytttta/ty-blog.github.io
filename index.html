2. 两数相加

Description:

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

提示：

  • 每个链表中的节点数在范围 [1, 100] 内
  • 0 <= Node.val <= 9
  • 题目数据保证列表表示的数字不含前导零

思路1：

将链表中每个节点的数据取出，求出实际值，然后两个链表的数据值相加，再解析成链表—不可实现，节点数量在1，100内，超过long存储最大值。

思路2:

将每个节点对应值相加，进位单独存储。

1. Two Sum

Description:

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly\ one solution, and you may not use the same element twice.

*Example *:

分析： 给定一个数组和一个target， 返回数组中两个元素的下标，这两个元素相加等于target。假设有且仅有一个解法，并且一个元素只能用一次。

方法1:

将所有元素两两相加，得到最终结果 复杂度O(n*n)

  • 时间复杂度较高

方法2:

使用map存储数据，循环遍历数组，如果target-nums[i] 在map中，返回对应value值，否则将当前值作为key，索引作为value存储在map中。 时间复杂度O(n)

方法 提交结果 运行时间 内存消耗 语言
方法2 通过 0 ms 38.6 MB Java
方法1 通过 0 ms 38.7 MB Java
  • 数据量较小，差距不大

1.1 安装git工具

Git是目前世界上最流行的分布式版本控制系统。使用Git可以帮助我们把本地的网页和文章等内容提交到Gihub上，实现同步。
下载地址：https://git-scm.com/downloads
测试安装：git - -version，查看显示版本号即成功。
