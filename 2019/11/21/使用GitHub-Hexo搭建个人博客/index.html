使用GitHub+Hexo搭建个人博客

效果展示： ty的博客[https://tytttta.github.io/ty-blog.github.io/]

一 环境准备

主要包括 安装git工具，搭建Node环境，安装Hexo工具，，注册git账号

1.1 安装git工具

Git是目前世界上最流行的分布式版本控制系统。使用Git可以帮助我们把本地的网页和文章等内容提交到Gihub上，实现同步。
下载地址：https://git-scm.com/downloads
测试安装：git - -version，查看显示版本号即成功。

1.2 搭建Node.js环境

一般常见的博客中会推荐在 下载地址：http://nodejs.cn/download 页面下载安装包，然后安装。但是通过这种方式安装，在1.3步会有一些小问题(会在1.3中阐述)，所以这里推荐使用NVM 即 node version manager安装，具体可见： https://github.com/nvm-sh/nvm#important-notes

本人使用的是git安装，由于本人电脑是mac系统，以下操作可能只适用于mac和Linux：
